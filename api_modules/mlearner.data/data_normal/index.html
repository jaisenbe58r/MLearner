data_normal

data_normal(mu=0, sd=1, n=100)

Generate a Normal data distribution.

Attributes

  • mu : int or float

    mean value.

  • sd : int or float

    standard deviation.

  • n : int

    number of data in the dataset.

Returns

  • data : Uniform data distribution.
